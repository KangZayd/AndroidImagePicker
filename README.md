AndroidImagePicker
==================


simply extend the ImagePickerActivity and implement the methods 

* getCorpCacheFile()
* getCamaraCacheFile()
* onCropSucceed(File file)
* onCropFailed();

and than you can Capture and Crop an Image with the Device Camera.
